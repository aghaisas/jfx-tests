MenuBar : menus renaming test.

Notion: This test is targeted for testing System menu bar on MacOS. It is implemented, by making standart JavaFX menuBar, switching useSystemMenu option to true/false, to be a system menu bar (grey menu bar on the top of the screen). On other platforms (like windows, and linux), this option shouldn't affect anyhow, so just press "PASS".

This test verifies that you can change name (text property) of menus in system menu on MacOS.

  • Definition: changing text (renaming), mean, that name of menu was changed from "some name" to "Renamed some name". So, prefix "Renamed" is added.
  • Definition: to check, that menu item was renamed, you need to verify, that prefix "Renamed" was added to mentioned menus.

Test case. Rename and move.

  • Click button "Rename".
  • Check, that {"Menu 0 SubMenu 0", "Menu 1", "Menu 2 SubMenu 2"} were renamed.
  • Set useSystemMenu property to "true".
  • Check, that {"Menu 0 SubMenu 0", "Menu 1", "Menu 2 SubMenu 2"} are still renamed.

Test case. Move and rename.

  • Click button "Reset".
  • Set useSystemMenu property to "true".
  • Click button "Rename".
  • Check, that {"Menu 0 SubMenu 0", "Menu 1", "Menu 2 SubMenu 2"} were renamed.

Notion

  • You can use "Reset" button to reset testing application.
