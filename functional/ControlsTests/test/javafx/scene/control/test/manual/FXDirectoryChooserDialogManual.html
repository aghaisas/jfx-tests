Directory chooser manual test

Description

JavaFX provide common access to native file and directory choosers. Directory chooser allows to select a single directory. We will test it.

Note

There is a "No errors" label in the bottom of the scene. If any error happens, in the place of this label will be shown an indication about error, and error will be printed into error output. Please, check this label, on each step, and fail the test, if you see any exception.

Step 1: Open/close case.

  • Press "Open Dir FX" button to see the directory chooser.
  • Close directory chooser.
  • Check that there are no exceptions, everything works fine, "no dir choosen" changed to message about "null" file.

Step 2: Directory choosing.

  • Open directory chooser via "Open Dir FX" button.
  • Select any folder you like. And press select.
  • Check that there are no exceptions, everything works fine, result points to the directory you've selected.

Step 3: Selecting of typed in folder name.

  • Open directory chooser via "Open Dir FX" button.
  • Type name of any folder, which you see in shown area (or navigate to any location, where folders are present) in an according field. And press select.
  • Check that there are no exceptions, everything works fine, result points to the directory you've selected.

Step 4: Title property checking.

Directory chooser can have a specified title.
  • Print "Custom title" in a text field for Title.
  • Open directory chooser via "Open Dir FX" button.
  • Check that that title of opened window is "Custom title" (if applicable).
  • Close dialog, and check, that there are no errors.
