MenuBar Events Test

Notion: This test is targeted for testing System menu bar on MacOS. It is implemented, by making standart JavaFX menuBar, switching useSystemMenu option to true/false, to be a system menu bar (grey menu bar on the top of the screen). On other platforms (like windows, and linux), this option shouldn't affect anyhow, so just press "PASS".

This test verifies that MenuBar Menu's events are received for MacOS.

Test case 1

  • Set useSystemMenu property to "true" by switching "Use system menu bar" toggle button on.
  • Clear events log by pressing "Clear Events" button.
  • Expand "Menu 0" in MacOS application menu.
  • Expand "Menu 0 SubMenu 0" in sub menu.
  • Expand "Menu 0 SubMenu 0 SubMenu 0" in sub menu.
  • Collapse all menus by clicking outside of menu's area.
  • Verify that log contains following:
      OnShowing event from Menu 0
      OnShown event from Menu 0
      OnShowing event from Menu 0 SubMenu 0
      OnShown event from Menu 0 SubMenu 0
      OnShowing event from Menu 0 SubMenu 0 SubMenu 0
      OnShown event from Menu 0 SubMenu 0 SubMenu 0
      OnHiding event from Menu 0 SubMenu 0 SubMenu 0
      OnHidden event from Menu 0 SubMenu 0 SubMenu 0
      OnHiding event from Menu 0 SubMenu 0
      OnHidden event from Menu 0 SubMenu 0
      OnHiding event from Menu 0
      OnHidden event from Menu 0
