Color Picker manual test.

The only step :

Custom color popup - is a popup, where you can select custom color. This window is resizable. And contains many controls. It is needed to check, that it is not broken during resizing, and all tabs are shown correctly.
  • Click on ColorPicker button.
  • Click on "Custom color..." hyperlink. Now, you see "Custom colors ..." dialog.
  • Having selected each of 3 tabs : "Web", "HSB", "RGB", try to resize dialog.
  • Resize it to be fully compressed, and very extended.
  • Check, that it resizes proportionally, without large gaps between controls and check, that it still functioning (responses on button clicks, fields receive focus, focus traversing on tab and arrows works).

Note that the main rectangle (in the upper left corner) will always remain square. Therefore there may be empty space below it.
