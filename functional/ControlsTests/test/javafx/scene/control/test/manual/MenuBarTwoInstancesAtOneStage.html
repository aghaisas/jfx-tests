MenuBar Add Menu Test

Notion: This test is targeted for testing System menu bar on MacOS. It is implemented, by making standard JavaFX menuBar, switching useSystemMenu option to true/false, to be a system menu bar (grey menu bar on the top of the screen). On other platforms (like windows, and linux), this option shouldn't affect anyhow, so just press "PASS".

This test verifies that stage has only one system menu.

Test case 1

  • Press "Use system menu bar" toggle button on the first panel
  • Press "Use system menu bar" toggle button on the second panel
  • Verify that system menu with prefix #1 is at the top
  • Unpress "Use system menu bar" toggle button on the first panel
  • Verify that there is no system menu at the top
  • Unpress "Use system menu bar" toggle button on the second panel

Test case 2

  • Press "Use system menu bar (using CSS)" on the second panel
  • Press "Use system menu bar (using CSS)" on the first panel
  • Verify that system menu with prefix #2 is at the top
  • Unpress "Use system menu bar (using CSS)" on the second panel
  • Verify that there is no system menu at the top
