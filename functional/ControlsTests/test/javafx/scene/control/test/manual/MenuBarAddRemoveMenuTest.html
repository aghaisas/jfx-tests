MenuBar Add Menu Test

Notion: This test is targeted for testing System menu bar on MacOS. It is implemented, by making standart JavaFX menuBar, switching useSystemMenu option to true/false, to be a system menu bar (grey menu bar on the top of the screen). On other platforms (like windows, and linux), this option shouldn't affect anyhow, so just press "PASS".

This test verifies that adding or removing menus to MenuBar work correctly for MacOS.

Test case 1

  • Set useSystemMenu property to "true".
  • Add new menu by pressing "Add Menu" button.
  • Verify that "Menu 3" with 3 levels of submenus has been added to MacOS application menu at the proper position.
  • Select several menu items of "Menu 3".
  • Verify that "Last selected item" label at the left-bottom corner of test application contains the name of the last selected menu item
  • Set useSystemMenu property to "false".
  • Verify that "Menu 3" has also been added to MenuBar.
  • Repeat for different positions.

Test case 2

  • Set useSystemMenu property to "true".
  • Remove menu by pressing "Remove menu" button.
  • Verify that menu at specified position has been removed.
  • Set useSystemMenu property to "false".
  • Verify that menu has been removed from MenuBar also.
  • Repeat for different positions.

Notion

  • You can use "Reset" button to reset testing application.
