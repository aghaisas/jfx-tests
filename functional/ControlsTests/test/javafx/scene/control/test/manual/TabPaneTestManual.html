Animated test

Notion: This test is targeted for testing of animation in TabPane. Actions of adding new tab, and removing existing tab can be animated or not. When you add new tab, and adding is animated, tab should appear smoothly, if it is not animated - at once. The same about removing. And we need to test switching of animated/not animated actions. For animated case - to check, that it is done well.

This test verifies that Animated property of tap pane can be switched, and quality animation itself.

Test case.

  • Change toggle button pressed state to see text "Animated".
  • Testing animated property, check, that animation is done or not done, according to animated property value.
  • Testing animated property, fail test if you see wrong, not smooth, strange animation, or graphical artifacts.
  • Add several tabs at 0, at the end, in the middle (setting according index into text field).
  • Remove tabs in the beginning and in the end, and in the middle.
  • Switch animated property to false, switching the toggle button pressed state.
  • Add several tabs at 0, at the end, in the middle (setting according index into text field).
  • Remove several tabs in the beginning and in the end, and in the middle.
