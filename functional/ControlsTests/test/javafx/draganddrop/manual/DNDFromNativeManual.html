This tests check native drag and drop support

Please do next steps

  1. From "content" folder in basedir of testsuite open file "dragboard.html" in "Mozilla Firefox" web browser
  2. For each content do following steps:
    1. In Target application (with title, describing current run options) choose checkboxes with approapriate data format(plain text, URL, image, files)
    2. Drag and drop content from web browser to button in red frame in "Target" app. For files use file browser. For URL don't have to select text of URL. Just drag and drop link.
    3. Verify content in "Transfered content" section. Text, images should be the same as in browser. For files you should get full paths on your machine. If content is different press fail
  3. Press pass in case of successful verification.
    Don't pay attention on lines between content descriptions on the right side of the scene. They are needed only for convenient view.
