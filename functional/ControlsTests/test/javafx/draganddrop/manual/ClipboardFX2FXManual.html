Clipboard from FX to FX transfer test

Step 1: Install runtime

  • If you had runtime installed from other FX sdk remove it through Windows Control Panel.
  • Otherwise please install runtime: go to SDKROOT and execute javafx-2.0-beta-windows-i586.exe
N.B.: note that you can install new runtime just once for each sdk version testing.

Step 2: Run apps

Run two commands from console from basedir of ControlsTests 
  • java -jar content/source.jar lightgreen onlySource
  • java -jar content/source.jar lightgray onlyTarget
You should see two windows: with title "Source" and another (target) stage with description of current run mode (rendering, build, etc).

Step 3: Verify Data formats

  • In each window choose the "Custom(bytes)", "Custom(string)", "Custom(class)" data formats.
  • Click "Put to clipboard" and then "paste from clipboard" buttons
  • Verify transfered content. You should see in transfered section of Target window:
    • bytes: "1, 2, 3, 4"
    • customString: "Hello Custom String!"
    • customClass: "Class{ clazz=Class{ clazz=null, i=100, d=100.9}, i=10, d=10.9}"
In case of successfull verification press pass, otherwise fail.
Don't pay attention on lines between content descriptions on the right side of the scene. They are needed only for convenient view.
