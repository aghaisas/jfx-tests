ScrollPane & TextArea Touch Test

Prerequisites: Installed FX runtime, multitouch hardware support

  • Uninstalled all previous version of FX sdk & runtime;
  • Installed FX runtime
  • Please see SceneGraphTests/README-mutitouch.html file to get basic knowledges about multitouch.

Test workflow:

  1. Take a look at the test application. It looks like:
  2. This test is applicable only for Windows 7 with touch screen. In other case press "Passed"
  3. Use touch screen and do scroll gesture outside ScrollPane & TextArea (e.g like on the screen-shot)
  4. Use mouse and set focus on TextArea
  5. Turn mouse wheel. Check TextArea scrolling. If TextArea does not scroll test fail
  6. Use touch screen and do scroll gesture on ScrollPane
  7. Use mouse and set focus on TextArea
  8. Turn mouse wheel. Check TextArea scrolling. If TextArea does not scroll test fail
  9. In case of successful verifications test pass
