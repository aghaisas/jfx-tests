LCD Text Test

This test verify set fullScreenProperty true in Mac OS X native full screen mode.

Test workflow:

  • Only for Mac OS X 10.7 or higher! In other platform test pass!
  • Enter to native full screen mode(button in top right corner)
  • Click on button "Set fullscreen true"
  • Exit from native full screen mode(button in top right corner)
  • Enter to native full screen mode again(button in top right corner)
  • Click on button "Set fullscreen true"
  • Exit from native full screen mode by press 'Esc'
  • There should be no errors.
