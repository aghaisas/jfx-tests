LCD Text Test

This test verify enter modality window to Mac OS X native full screen mode.

Test workflow:

  • Only for Mac OS X 10.7! In other platform test pass!
  • Click on button "Show modality window"
  • Enter modality window to native full screen mode(button in top right corner)
  • New window must be opened in native full screen mode.
  • Exit from native full screen mode(button in top right corner)
  • You should see a modal window in normal mode(you should see the first window)
  • Close modality window.
  • You should see the first window in normal mode.
  • There should be no errors.
