This test verifies that menus work properly in Mac OS X native full screen mode.

Test workflow:

  • Only for Mac OS X 10.7 or higher! Please mark the test as PASS on other platforms.
  • Enter native full screen mode.
  • Try to show and hide 'Menu1'. Messages 'Menu 1 shown' and 'Menu 1 hidden' should be added to text area.
  • Click on 'Menu Item 1' in 'Menu1'. Messages 'Menu 1 shown', 'Menu 1 hidden' and 'Menu Item 1 Action' should be added to text area.
  • Press 'Ctrl + M'. Message 'Menu Item 1 Action' should be added to text area.
  • Press 'Use system menu' button.
  • Try to show and hide 'Menu1'. Messages 'Menu 1 shown' and 'Menu 1 hidden' should be added to text area.
  • Click on 'Menu Item 1' in 'Menu1'. Messages 'Menu 1 shown', 'Menu 1 hidden' and 'Menu Item 1 Action' should be added to text area.
  • Press 'Ctrl + M'. Message 'Menu Item 1 Action' should be added to text area.
  • Exit native full screen mode.
  • There should be no errors.
