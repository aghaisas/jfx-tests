This test verifies modal windows work properly in Mac OS X native full screen mode.

Test workflow:

  • Only for Mac OS X 10.7 or higher! Please mark the test as PASS on other platforms.
  • Enter native full screen mode.
  • Click "Show modality window" button.
  • New window must be opened in native full screen mode.
  • Exit native full screen mode.
  • You should see the modal window in normal mode; you should not see the first window.
  • Close the modal window.
  • You should see the first window in native full screen mode.
  • There should be no errors.
