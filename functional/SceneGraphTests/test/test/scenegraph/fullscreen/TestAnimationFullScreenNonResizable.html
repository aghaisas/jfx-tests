This test verifies animation on enter/exit full screen mode works properly for non resizable window.

Test workflow:

  • Only for Mac OS X 10.7 or higher! Please mark the test as PASS on other platforms.
  • On Mac OS X 10.10 you should be unable to enter full screen mode (the button should be disabled).
  • For Mac OS X below 10.10 (10.9, 10.8 etc):
    • Enter native full screen mode.
    • The window should move to the gray background
    • During the animation window should not be stretched.
    • Exit native full screen mode.
    • The window should move back.
    • During the animation window should not be stretched.
  • There should be no errors.
