Image loading manual test

Step 1: Install runtime

  • If you had runtime installed from other FX SDK remove it.
  • Download JDK cobundle.
  • FX runtime should be installed with JDK cobundle.
N.B.: note that you can install new runtime just once for each sdk version testing.

Step 2: Run app

Take a look at the application which presents same image on black and white backgrounds.
Please open the following file animated.html and use following image for comparison: infinite_loop.gif with number 11
Look at the images and verify following items:
  • image loaded
  • images in browser and application looks similar except that the background area width in browser could differs from background area width in application(in app background area width is the same as image width, and in browser NOT). Take an account that image can have transparent areas.
  • if images are animated verify that animations looks equals. Please note that it's possible that animation playback can be asynchronius (shifted) because of different loading time.
  • note that background in app has the same width as image, and image background in browser has the same width as table cell. So it may be different and it's OK.

    • In case of successful verification press pass, otherwise fail.
