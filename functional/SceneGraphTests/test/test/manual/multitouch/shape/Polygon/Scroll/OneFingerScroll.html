Multitouch Shape Test: Polygon; One-finger Scroll gesture

Prerequisites: Installed FX runtime, multitouch hardware support

  • Uninstalled all previous version of FX sdk & runtime;
  • Installed FX runtime
  • Please see SceneGraphTests/README-mutitouch.html file to get basic knowledges about multitouch.

Test workflow:

  1. Take a look at the test application (MultiTouchApp)
  2. This test is applicable only for touch screen. In other case press "Passed"
  3. Push toggle button "Direct"
  4. Use Node.addEventHandler to receive multitouch events:
    1. Select "ADD_HANDLER" (it is default value) in the choice box
    2. Push button "AddPolygon"
    3. Check that a random polygon is painted on the scene
    4. Do One-finger Scroll gesture for the shape.
    5. In case shape stays half-transparent after gesture start - test fail
    6. In case the shape doesn't drag (go after the finger) during gesture - test fail
    7. In case shape is not half-transparent after gesture end (no touches) - test fail
    8. Push button "ClearScene"
  5. Use Node.addEventFilter to receive multitouch events:
    1. Select "ADD_FILTER" in the choice box
    2. Push button "AddPolygon"
    3. Check that a random polygon is painted on the scene
    4. Do One-finger Scroll gesture for the shape.
    5. In case shape stays half-transparent after gesture start - test fail
    6. In case the shape doesn't drag (go after the finger) during gesture - test fail
    7. In case shape is not half-transparent after gesture end (no touches) - test fail
    8. Push button "ClearScene"
  6. Use Node.setOn<EventType> handler to receive multitouch events:
    1. Select "SET_ON_EVENT_HANDLER" in the choice box
    2. Push button "AddPolygon"
    3. Check that a random polygon is painted on the scene
    4. Do One-finger Scroll gesture for the shape.
    5. In case shape stays half-transparent after gesture start - test fail
    6. In case the shape doesn't drag (go after the finger) during gesture - test fail
    7. In case shape is not half-transparent after gesture end (no touches) - test fail
  7. In case of successful verification press pass, otherwise fail
