Multitouch Shape Test: Text, Zoom / Rotate gesture, null fill

Prerequisites: Installed FX runtime, multitouch hardware support

  • Uninstalled all previous version of FX sdk & runtime;
  • Installed FX runtime
  • Please see SceneGraphTests/README-mutitouch.html file to get basic knowledges about multitouch.

Test workflow:

  1. Take a look at the test application (MultiTouchApp)
  2. If test runs on the computer with touch screen push toggle button "Direct" (for touch pad using this button must be unselected)
  3. Push button "AddText"
  4. Check that the text "Java FX" is painted at the center of the scene
  5. Double click on the shape
  6. Check that shape fill is changed to null (shape becomes not )
  7. Push toggle buttons "Log ZoomStart/Finish" & "Log RotateStart/Finish"
  8. Do Zoom / Rotate gesture the shape. Be sure that center between touches is inside the text rectangle and that center is not over the text symbols
  9. In case shape stays half-transparent after gesture start - test fail
  10. In case shape is not scaled during gesture - test fail
  11. In case shape is not rotated during gesture - test fail
  12. In case shape is not half-transparent after gesture end (no touches) - test fail
  13. In case of successful verification press pass, otherwise fail
