Multitouch Inertia Test

Prerequisites: Installed FX runtime, multitouch hardware support

  • Uninstalled all previous version of FX sdk & runtime;
  • Installed FX runtime
  • Please see SceneGraphTests/README-mutitouch.html file to get basic knowledges about multitouch.

Test workflow:

  1. Take a look at the test application (MultiTouchApp)
  2. If test runs on the computer with touch screen push toggle button "Direct" (for touch pad using this button must be unselected)
  3. Push any button "AddArc", "AddCircle", etc to add a shape on the scene
  4. Do Scroll gestures for the shape. Your fingers must have velocity during release the touch screen/touch pad
  5. In case the shape continues the motion after gesture end - test fail
  6. Push toggle button "Inertia"
  7. Do Scroll gestures for the shape. Your fingers must have velocity during release the touch screen/touch pad
  8. In case the shape does not continue the motion after gesture end - test fail
  9. In case of successful verification press pass, otherwise fail
