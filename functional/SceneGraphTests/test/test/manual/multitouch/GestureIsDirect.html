Multitouch Direct / Indirect Test

Prerequisites: Installed FX runtime, multitouch hardware support

  • Uninstalled all previous version of FX sdk & runtime;
  • Installed FX runtime
  • Please see SceneGraphTests/README-mutitouch.html file to get basic knowledges about multitouch.

Test workflow:

  1. Take a look at the test application (MultiTouchApp)
  2. Push any button "AddArc", "AddCircle", etc to add a shape on the scene
  3. Skip the following steps if the operating system does not Windows 7 or there is not touch screen
    1. Do any Swipe, Zoom/Rotate & Scroll gestures for the shape.
    2. In case the shape's colors are changed, shape drags or scaled / rotated - test fail
    3. Push toggle button "Direct"
    4. Do any Swipe, Zoom/Rotate & Scroll gestures for the shape.
    5. In case the shape's colors aren't changed, shape doesn't drag or not scaled / rotated - test fail
  4. Skip the following steps if the operating system does not Mac OS X or there is not touch pad
    1. Do any Zoom/Rotate & Scroll gestures for the shape.
    2. In case the shape doesn't drag or not scaled / rotated - test fail
    3. Push toggle button "Direct"
    4. Do any Swipe, Zoom/Rotate & Scroll gestures for the shape.
    5. In case the shape drags or scaled / rotated - test fail
  5. In case of successful verification press pass, otherwise fail
