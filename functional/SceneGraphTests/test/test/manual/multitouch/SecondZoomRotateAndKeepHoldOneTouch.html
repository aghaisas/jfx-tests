Multitouch Test: second zoom/rotate gesture with hold & keep one touch

Prerequisites: Installed FX runtime, multitouch hardware support

  • Uninstalled all previous version of FX sdk & runtime;
  • Installed FX runtime
  • Please see SceneGraphTests/README-mutitouch.html file to get basic knowledges about multitouch.

Test workflow:

  1. This test is applicable only for touch screen. In other case press "Passed"
  2. Take a look at the test application (MultiTouchApp)
  3. Push toggle button "Direct"
  4. Push any button "AddArc", "AddCircle", etc to add a shape on the scene
  5. Check that a random shape is painted on the scene
  6. Press and hold one touch outside the shape
  7. With the other finger do zoom/rotate outside the shape. All touches must be outside the shape too
  8. Release second touch
  9. Press second touch so that do Zoom / Rotate gesture for the shape (second touch must be outside the shape). Do Zoom / Rotate gesture for the shape
  10. In case the shape is not scaled / rotated - test fail
  11. In case of successful verification press pass, otherwise fail
