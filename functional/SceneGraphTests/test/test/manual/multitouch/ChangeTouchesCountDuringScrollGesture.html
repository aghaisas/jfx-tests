Multitouch Test: Change touches count during scroll gesture

Prerequisites: Installed FX runtime, multitouch hardware support

  • Uninstalled all previous version of FX sdk & runtime;
  • Installed FX runtime
  • Please see SceneGraphTests/README-mutitouch.html file to get basic knowledges about multitouch.

Test workflow:

  1. This test is applicable only for touch screen. In other case press "Passed"
  2. Take a look at the test application (MultiTouchApp)
  3. Push toggle button "Direct"
  4. Add to scene two random shapes (use for it toggle buttons "AddArc", "AddCircle", ... etc )
  5. Push toggle button "Log ScrollStart/Finish"
  6. Do One-finger Scroll gesture for the shape 1. And keep hold this touch
  7. Do Two-finger Scroll gesture for the shape 1
  8. In case there are not log messages in TextArea like below, test fail I.e. After start second gesture SCROLL_FINISHED & SCROLL_STARTED must be recieved 
    ScrollEvent [..., eventType = SCROLL_STARTED, ...]
ScrollEvent [..., eventType = SCROLL_FINISHED, ...]                
ScrollEvent [..., eventType = SCROLL_STARTED, ...]
ScrollEvent [..., eventType = SCROLL_FINISHED, ...]
  9. Do One-finger Scroll gesture for the shape 1. And keep hold this touch
  10. Do Two-finger Scroll gesture for the shape 2.
  11. In case shape 1 continue scroll (drag) during second scroll gesture - test fail
  12. In case shape 2 doesn't scroll (drag) during second scroll gesture - test fail
  13. In case of successful verification press pass, otherwise fail
