Touch Point Grab API Test

Prerequisites: Installed FX runtime, multitouch hardware support

  • Uninstalled all previous version of FX sdk & runtime;
  • Installed FX runtime
  • Please see SceneGraphTests/README-mutitouch.html file to get basic knowledges about multitouch.

Test workflow:

  1. This test is applicable only for touch screen. In other case press "Passed"
  2. Take a look at the test application (TouchPointGrabApp)
  3. There are three columns with labels "Test1", "Test2", "Test1" & buttons "Begin" / "End" for each column
  4. Push the button "Begin" in the "Test1" column
  5. Do Scroll gestures on the green rectangle
  6. Push the button "End" in the "Test1" column
  7. In case label below the button "End" in the first column is "Test1 failed" - test failed
  8. Push the button "Begin" in the "Test2" column
  9. Do Scroll gestures on the green rectangle
  10. Push the button "End" in the "Test2" column
  11. In case label below the button "End" in the second column is "Test2 failed" - test failed
  12. Push the button "Begin" in the "Test3" column
  13. Do Scroll gestures on the green rectangle
  14. Push the button "End" in the "Test3" column
  15. In case label below the button "End" in the third column is "Test3 failed" - test failed
  16. Push the button "Begin" in the "Test4" column
  17. Do Scroll gestures from the green rectangle, over pink & black rectangle to yellow rectangle. And and gesture on the yellow rectangle. The route of scroll gesture must be like as the red line on the figure below:

  18. Push the button "End" in the "Test4" column
  19. In case label below the button "End" in the fourth column is "Test4 failed" - test failed
  20. In case of successful verification press pass, otherwise fail
