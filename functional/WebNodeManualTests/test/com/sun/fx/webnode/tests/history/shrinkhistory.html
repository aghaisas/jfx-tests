Testing shrinking history to a smaller size

  1. Check that the following text can be found on the screen:
    Current index: 0
    Current number of entries: 1
    Page 1
  2. Click "Next page" link. Check that the following text can be found on the screen:
    Current index: 1
    Current number of entries: 2
    Page 2
  3. Click "Set history size to 1" button. Check that the following text can be found on the screen:
    Current index: 0
    Current number of entries: 1
  4. Click "Next page" link. Check that the following text can be found on the screen:
    Current index: 0
    Current number of entries: 1
    Page 3
If both steps are successful, the test passes.
