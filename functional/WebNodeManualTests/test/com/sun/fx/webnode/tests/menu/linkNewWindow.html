Testing "Open Link in New Window" item in context menu

  1. Wait for Page 1 to load. Bold text "Page 1" should appear.
  2. Right-click on "To Page 2" link. Context menu should appear.
  3. Click on "Open Link in New Window" menu item. A new page, with bold text "Page 2" header, should load below, replacing green section of application window.
The test passes if all steps pass.
