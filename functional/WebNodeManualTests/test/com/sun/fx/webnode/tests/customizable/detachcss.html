Attaching and detaching user stylesheet

  1. Check that "This is green with red background when user CSS is attached" line has black font on white background.
  2. Click on "Attach user stylesheet". Font of "This is green with red background when user CSS is attached" should become green, and background should become red.
  3. Click on "Detach user stylesheet". Font of "This is green with red background when user CSS is attached" should become black, and background should become white.
If all steps pass, the test passes.
