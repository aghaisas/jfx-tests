Test for dragenter and dragleave events

  1. Wait for the test page to load.
  2. Start dragging the "Drag me!" rectangle. You should be able to drag it.
  3. Drag the rectangle ontop of "Drag area". Do not drop the rectangle. The color of drag area should turn to light red.
  4. Drag the rectangle away from drag area. The color of drag area should turn to light green.
The test passes if every step passes.
