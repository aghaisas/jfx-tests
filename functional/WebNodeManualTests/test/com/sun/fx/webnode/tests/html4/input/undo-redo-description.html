The following keys denote undo/redo actions on different systems:
OS undo redo
Linux Ctrl+Z Ctrl+Y
Windows Ctrl+Z Ctrl+Shift+Z
Mac OS Cmd+Z Cmd+Shift+Z
For every type of input do the following:
  1. type "abacaba"
  2. press undo combination for your OS
  3. check that the text is removed
  4. press redo combination for your OS
  5. check that the text is back again
  6. add "qwertyuiop" to the text already there
  7. press undo combination for your OS
  8. check that appended text is removed
  9. press redo combination for your OS
  10. check that it's back again
  11. press undo combination for your OS twice
  12. check that the field is empty
