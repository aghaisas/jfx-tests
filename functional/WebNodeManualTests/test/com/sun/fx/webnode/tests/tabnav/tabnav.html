Test for navigating between the links using "Tab" key.
  1. Wait for page with "Page 1" title to be loaded. Click on Launched window so that it becomes focused.
  2. Press "Tab" key. Make sure that the first link becomes focused.
  3. Press "Tab" key again. Make sure that the second link becomes focused.
  4. Press "Tab" key again. Make sure that the third link becomes focused.
  5. Hold "Shift" key and press "Tab" key. Make sure that the second link becomes focused.
  6. Hold "Shift" key and press "Tab" key again. Make sure that the first link becomes focused.
  7. Press "Tab" key until "To page 2" link becomes focused.
  8. Press "Enter" key. Make sure that another page with "Page 2" title has loaded.
Test passes if all cases pass.
