Testing "Stop Page Loading" item in context menu

  1. Wait for Page 1 to load. Bold text "Page 1" should appear.
  2. Check that an image of a horse near "Horse" text is rendered. This means that the server with test is accessible.
  3. Click on "Large document" link. A new page with 10 heavy images will start loading. Note: you should perform next 2 steps quickly.
  4. Right-click in white area to the right of the images. Context menu will appear.
  5. Click on "Stop loading" item. Page loading should stop, and the images should also stop loading.
  6. Wait some more time to be sure that the page won't continue loading. About 5 minutes should be enough.
  7. Scroll the document down. The text at the bottom should say "Test passed".
The test passes if all steps pass.
