Test for uploading a file after clicking cancel in file select window

  1. Wait for test page to load. The application with loaded page should look like below (directory where files were generated can be different):

  2. Click on "Choose File" button. A modal window with file selection should open.

  3. Select file "test1.txt", and press "Open" button.
    Modal window with file selection should close, and "test1.txt" text should appear next to "Choose File" button.
  4. Click on "Choose File" button again.

    Click on "Cancel" button.

  5. Click on "Upload" button.
    The following text should appear:
    Files successfully uploaded:
