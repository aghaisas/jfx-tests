Testing "Copy Link to Clipboard" item in context menu

  1. Wait for Page 1 to load. Bold text "Page 1" should appear.
  2. Right-click on "Qilin" link. Context menu should appear.
  3. Click on "Copy Link to Clipboard" menu item.
  4. Paste clipboard contents to text field (marked with "<-") using either shortcut or context menu. Text field should now contain the following link:
    http://en.wikipedia.org/wiki/Qilin
  5. Paste clipboard contents to rich text editor. Rich text editor should now contain the following link:
    Qilin
    It should have blue color.
  6. Click on the link pasted to rich text editor. Wikipedia page about Qilin should load.
The test passes if all steps pass.
