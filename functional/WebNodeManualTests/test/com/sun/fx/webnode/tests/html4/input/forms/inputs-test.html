Submitting checkboxes and radio buttons

Submitting check boxes with different name
Check any checkboxes, verify that the page with the word "Success" was loaded after submitting the form.
Also check that for every checkbox you've selected there will be the text before checkbox you've selected followed by "=true" in the page URL on that page.



Submitting check boxes with same names
Check any checkboxes, verify that the page with the word "Success" is loaded after submitting the form.
Also check that for every checkbox you've selected there will be "option=" followed text before checkbox you've selected in the page URL on that page.



Submitting the disabled checkbox
Check that the page with the word "Success" is loaded after submitting the form.
Also check that page URL on that page doesn't contain "option=AAA" string.

Submitting radio buttons
Check that the page with the word "Success" is loaded after submitting the form.
Also check the page URL ends up with "submit_action.html?option=" and text before selected radio button.



Submitting radio buttons with disabled elements
Check that the page with the word "Success" is loaded after submitting the form.
Also check the page URL ends up with "submit_action.html?option=" and text before selected radio button.
Use reset button to change the controls state to default.



Submitting radio buttons with disabled active element
Check that the page with the word "Success" is loaded after submitting the form.
Also check that:
  • the page URL ends up with "submit_action.html" or "submit_action.html?" if you haven't changed the state of controls below (left the disabled radio button checked).
  • the page URL ends up with "submit_action.html?option=" and text before selected radio button if you've changed the state of controls below (checked another radio button).
Use reset button to change the controls state to default (disabled radio button selected).
